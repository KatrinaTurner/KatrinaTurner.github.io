NxBook - Teacher Dashboard

NxBook is a tool that allows educators to develop teaching modules in Jupyter Notebooks.

Overview

NxBook is a tool that integrates Javascript widgets with Jupyter Notebook and allows educators to develop teaching modules in Jupyter Notebooks. My main focus for my Master's Project is to create a full stack React Web App "teacher dashboard" for the educators to manage their modules and view data analytics on their students' progress.
